Cryptography

"the art of writing or solving codes"

  • Definitions

    1. Encryption: The process of obfuscating data for unwanted others.

    2. Decryption: The process of de-obfuscating data
      that has been encrypted.

    3. Cipher: A way to write something in an encrypted fashion.

    4. Plaintext: Text that is written in a non-encoded way.

    5. Ciphertext: Text that is written in an encoded way.

    6. Key: A tool used to encrypt data.

    7. Private Key: A way to write to decrypt a message that was encrypted by a public key

    8. Public Key: A way to encrypt data, by anyone.

    9. Digital Signature: A digitally produced tag used for encrypting data.

    10. One Time Pad: A form of encryption where all decoded values are equally likely to be the original.

    11. Seed: The start to generating a pseudorandom number.

    12. xor: Is a bitwise operator that produces true if a or b is true.

    13. Bitwise: An operation that changes the bits in a binary representation of a datum.
